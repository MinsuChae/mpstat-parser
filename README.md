#mpstat parser

##개발 환경

###개발 언어
 * JAVA

###개발 환경
 * JDK8
 * ECLIPSE
JDK8에서 테스트하였으나 JDK8 에서 나온 라이브러리만 사용한 것이 아니기 때문에 일부 하위버전도 호환

###실행 조건
 * JRE8
JRE8 환경으로 런타임 라이브러리를 설정하였으나 위와 같은 이유로 하위 버전에서 재빌드하면 실행 가능

##개발 목적
자바 언어 복습 겸 벤치마킹할시 [mpstat](https://ko.wikipedia.org/wiki/Mpstat) 로 여러번 측정시 원하는 자료 추출의 어려움으로 인해 개발

여기에 업로드 한 것 이유중 하나는 git 툴 공부할 목적이다.

##사용 방법

 1. mpstat 의 출력을 가지고 파일들을 만든다.
 2. 추출을 원하는 파일들로만 하나의 폴더에 모은다.
 3. 실행해서 원하는 CPU 번호를 입력한다. (0, 1, ALL 등)
 4. rowName 입력하는 곳에 추출을 원하는 항목을 입력한다.(%idle 등)
 5. Open Dir 통해 추출을 원하는 파일들이 모여있는 폴더를 선택한다.
 6. Process 를 클릭하면 해당하는 항목이 중앙 입력창에 전부 모인다.
 7.원하는 목적으로 해당 데이터들을 사용한다.


##질문 & 답
